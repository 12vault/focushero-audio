# focushero-audio

Audio assets for the FocusHero iOS app: ambient focus sounds and noise tracks.

Unless otherwise noted, all audio files in this repository are released under the CC0 1.0 Public Domain Dedication.

If you have focus tracks you’d like to suggest or produce, feel free to open an issue or a pull request with links, ideas, or audio files that you are happy to release under CC0. We’ll list contributors in this repository (and, where feasible, in the app) to credit people whose sounds make it into FocusHero.
